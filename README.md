# 技能五子棋 - 赖子版对战

一款融合传统五子棋与创新技能系统的多人在线对战游戏。

## 🎮 游戏特色

- **6种技能随机组合**：每局选择3个技能进入战场
- **实时技能对抗**：动态使用技能改变局势
- **匿名弹幕社交**：游客免登录观战并发送弹幕
- **荒诞赖子玩法**：掀桌、时光倒流等搞怪技能

## 🛠 技能列表

1. **飞沙走石** (💨) - 移除对手任意1枚棋子
2. **力拔山兮** (🔥) - 掀翻棋盘，强制平局
3. **静如止水** (❄️) - 冻结对手10秒
4. **两级反转** (🔄) - 翻转棋盘180°
5. **时光倒流** (⏰) - 双方各退一步
6. **暗度陈仓** (👻) - 下一次可覆盖棋子

## 🚀 快速开始

### 安装依赖
\`\`\`bash
pnpm install
\`\`\`

### 启动服务
\`\`\`bash
pnpm run dev
\`\`\`

前端将运行在 http://localhost:5173

## 📱 功能说明

### 用户系统
- **匿名准入**：无需注册，自动分配随机用户名
- **身份标识**：显示IP归属地（省份）和浏览器图标
- **会话保持**：使用localStorage保存临时身份

### 游戏模式
- **随机对局**：5秒内匹配对手
- **围观对局**：观看进行中的对局，发送弹幕
- **历史对局**：查看最近100局的对局记录

### 技能系统
- 每局拥有3个技能点
- 每个技能有使用次数限制和冷却时间
- 技能释放时有全屏特效

### 随机事件
- 每15手触发一次"天降正义"
- 随机在空位落1子

## 🎨 UI设计

- **主色调**：高饱和度紫绿撞色
- **荒诞精神**：中二口号、夸张动效
- **响应式布局**：完美适配PC和移动端

## 💾 数据存储

- 所有数据存储在内存中
- 在线用户、游戏状态、历史对局（最近100局）
- 适配无服务器环境（Vercel等）

## 📝 技术栈

- **前端**：Vue 3 + Canvas
- **后端**：Node.js + Express + Socket.IO
- **实时通信**：WebSocket
- **构建工具**：Vite

## 🌈 游戏玩法

1. 进入游戏，自动分配身份
2. 点击"随机对局"，选择3个技能
3. 等待匹配对手（5秒内）
4. 轮流落子，使用技能
5. 达成五连获胜
