# Counter-Strike: Web Edition

一个基于 Vue 3 的 CS:GO 风格第一人称射击网页游戏。

## 功能特性

### 🎮 游戏玩法
- **第一人称射击**: 完整的3D第一人称视角
- **武器系统**: 三种武器可供选择（AK-47、AWP、沙漠之鹰）
- **AI敌人**: 智能敌人自动移动和对战
- **回合制对战**: CT vs T 对战模式
- **实时计分**: 击杀敌人获得分数，先到5分获胜

### 🎯 游戏控制
- **W/A/S/D** - 前进/左移/后退/右移
- **鼠标移动** - 视角控制
- **左键点击** - 射击
- **R键** - 换弹
- **1/2/3键** - 切换武器
- **ESC键** - 暂停菜单

### 🎨 UI设计
完全按照CS:GO风格设计：
- **配色**: 深蓝色/海军蓝主色调，橙色强调色
- **字体**: Roboto, Montserrat 无衬线字体
- **风格**: 科技感、军事工业风、硬朗线条
- **布局**: 模块化、清晰的信息层级

### 📊 游戏界面
1. **主菜单**
   - 开始游戏
   - 设置（玩家名称、鼠标灵敏度、音量）
   - 统计数据（胜场、击杀、K/D比率等）
   - 游戏说明

2. **游戏界面**
   - HUD显示（生命值、护甲、弹药）
   - 准心
   - 计分板（CT vs T）
   - 击杀提示
   - 小地图
   - 武器选择器

3. **暂停菜单**
   - 继续游戏
   - 设置
   - 返回主菜单

4. **游戏结束界面**
   - 最终统计数据
   - 再玩一次
   - 返回主菜单

### 💾 数据存储
- **玩家设置**: 使用 localStorage 存储（玩家名称、灵敏度、音量）
- **游戏统计**: 使用后端内存存储（重启后重置）
  - 胜场/败场
  - 总击杀/总死亡
  - 准确率
  - 排名等级

### 🔧 技术栈
- **前端**: Vue 3 (Composition API)
- **路由**: Vue Router 4
- **HTTP**: Axios
- **后端**: Express.js
- **数据存储**: 内存 Map（符合无服务器部署要求）

### 🎖️ 排名系统
游戏包含CS:GO风格的排名系统：
- Silver I (0-4胜)
- Silver Elite (5-9胜)
- Gold Nova (10-19胜)
- Master Guardian (20-29胜)
- Legendary Eagle (30-39胜)
- Supreme Master (40-49胜)
- Global Elite (50+胜)

## 游戏特点

### ✨ 视觉效果
- 3D第一人称视角渲染
- 动态准心
- 血量和护甲进度条
- 击杀提示动画
- 小地图实时位置显示
- 武器换弹动画提示

### 🎯 游戏机制
- 真实的射击命中判定
- 弹药管理（弹夹/备弹）
- 自动换弹（弹夹空时）
- 准确率统计
- 回合倒计时（115秒）
- 多武器系统（不同伤害和射速）

### 🏆 统计追踪
游戏自动追踪以下数据：
- 击杀/死亡数
- K/D比率
- 射击准确率
- 胜/败场次
- 当前排名

## API接口

### 获取玩家统计
```
GET /api/player/stats
```

### 保存游戏结果
```
POST /api/player/stats
Body: { kills, deaths, accuracy, won }
```

### 重置统计数据
```
POST /api/player/reset
```

### 获取排行榜
```
GET /api/leaderboard
```

## 注意事项

1. **内存存储**: 所有游戏数据存储在服务器内存中，重启后会丢失
2. **单人模式**: 当前版本是单人对战AI模式
3. **浏览器兼容**: 需要支持 Pointer Lock API 的现代浏览器
4. **最佳体验**: 建议使用鼠标进行游戏

## 开发信息

- 端口: 5173 (前端), 3002 (后端)
- 代理: 前端通过 Vite 代理访问 /api
- 热更新: 支持开发时热更新
